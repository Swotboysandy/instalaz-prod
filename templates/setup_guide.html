Quick Start

Get Instalaz running in 2 minutes:

pip install -r requirements.txt
python app.py

Visit http://localhost:5000/onboarding to begin setup.

Setup

1. Create Facebook App

  1. Go to Facebook Developers
  2. Click "Create App" → Select "Business"
  3. Name your app (e.g., "My Instalaz")
  4. Add "Instagram" product

2. Configure OAuth

Important This step prevents connection errors.
  1. Go to Use Cases → Authentication → Settings
  2. Add redirect URI: http://localhost:5000/auth/callback
  3. Click Save

3. Get Credentials

  1. Settings → Basic
  2. Copy App ID
  3. Click "Show" → Copy App Secret

Connect Instagram

  1. Open the onboarding wizard
  2. Enter your App ID and App Secret
  3. Click "Connect with Facebook"
  4. Authorize permissions
  5. Done!
Note Your Instagram must be a Business Account connected to a Facebook Page.

Auto Posting

Instalaz can automatically post:

  • Carousels - 2-10 images per post
  • Reels - Short videos
  • Stories - Temporary content

Add Content

  1. Dashboard → Edit account
  2. Set image/video URLs
  3. Add caption URL (one caption per line)
  4. Save

Scheduling

Set posting times in the Schedule page:

{
  "morning": {"hour": 9, "minute": 0},
  "afternoon": {"hour": 15, "minute": 30},
  "evening": {"hour": 19, "minute": 0}
}

Multiple Accounts

Manage unlimited Instagram accounts:

  1. Click "Add Account"
  2. Enter account details
  3. Connect via Facebook OAuth
  4. Set unique schedule per account

Common Errors

"Insecure connection detected"

Fix: Add http://localhost:5000/auth/callback to OAuth Redirect URIs in your Facebook App.

"No Instagram accounts found"

Fix: Convert Instagram to Business Account and connect to a Facebook Page.

"Token expired"

Fix: Tokens last 60 days. Click "Reconnect" to get a new token.

Token Management

Instalaz automatically:

  • Exchanges short-lived tokens (2 hours) for long-lived tokens (60 days)
  • Tracks expiration dates
  • Notifies you when tokens need renewal
Best Practice Reconnect your account every 50 days to avoid expiration.
