# Demo Data for Instalaz

This directory contains placeholder content for demonstration and investor showcase purposes.

## Contents

### Images (`images/`)
Sample images for carousel posts sourced from free stock photo providers:
- High-quality lifestyle and business stock photos
- Properly sized for Instagram (1080x1080 or 1080x1350)
- No copyright restrictions

### Videos (`videos/`)
Sample videos for reels sourced from free stock video providers:
- 9:16 aspect ratio for Instagram Reels
- Duration: 15-30 seconds
- High quality, royalty-free content

### Captions (`captions.txt`)
Example captions for posts:
- Engaging, professional sample text
- Hashtag examples
- Call-to-action examples

## Free Resources Used

All demo content is sourced from free, open-source platforms:
- **Images**: [Unsplash](https://unsplash.com) - Free high-quality photos
- **Videos**: [Pexels](https://pexels.com/videos) - Free stock videos
- **Placeholder Data**: No real user accounts or personal information

## Usage

This demo data is automatically used when:
1. No real accounts are connected
2. The app is in showcase/demo mode
3. For testing new features

**Note**: Replace this demo content with your own images/videos when setting up real Instagram automation.
